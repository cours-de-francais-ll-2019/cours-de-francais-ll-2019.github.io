Les verbes 動詞どうし

Les types de verbes

  • Les verbes d’action : manger 食べるたべる
  • Les verbes d’état : devenir 成るなる + attribut du sujet 属性ぞくせい

Les formes de verbes

  • La forme active 能動態のうどうたい : Il allume la lumière.
  • La forme passive 受動態じゅどうたい : La lumière est allumée.

Allumer : 点けるつける
La lumière : 電気でんき

Les temps 時間じかん

  • Le passé 過去かこ
  • Le présent 現在げんざい
  • Le futur 未来みらい

La concordance des temps est importante !
